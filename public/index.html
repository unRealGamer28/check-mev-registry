Check Validator MEV Registry

A simple website to check if your validator is registered with MEV relays.

Enter your validator's public wallet address:

This website supports checking MEV registry for 6/7 public relays (Manifold currently doesn't support this check).

  • Flashbots
  • BloXroute Max Profit (Censorship-Free)
  • BloXroute Ethical (Censorship-Free)
  • BloXroute Regulated
  • Blocknative
  • Eden

Data Privacy: This site does not retain any data submitted by the user. No information about a specific validator or user are retained after data retrieval. All data displayed is public data from the blockchain.
